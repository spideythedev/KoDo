The Event Loop

JavaScript is strictly single-threaded. It can only execute one piece of code at a time on its main thread. So how does it handle complex tasks like fetching database records without freezing the entire application? It uses the Event Loop.

Micro vs Macro Tasks

When you run asynchronous code, it doesn't execute immediately. It gets pushed to a waiting area. Promises (like fetch) go to the high-priority Microtask Queue, while functions like setTimeout go to the lower-priority Macrotask Queue.
